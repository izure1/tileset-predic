학습 속도가 느릴 경우

	1. chunk 사이즈가 너무 작거나, 소스 이미지가 너무 크지 않은지.
	2. 하나의 소스 이미지가 너무 크지 않은지. 커다란 소스 이미지 한개보다 자그만한 소스 이미지 여러개가 학습 성능 시 더 좋다.


생성 속도가 느린 경우

	1. attempt 수치와 maxAttempt 수치가 높을 때 (데이터가 부족하기에 반복 생성 시도가 이유)
	2. dense시 sim, sensitive 수치가 낮을 경우 (많은 타일을 참조하기 때문에)


학습 데이터 부족

	1. 공백이 많을 경우
	2. 생성 속도가 느릴 경우 (반복 생성 시도 때문에)
